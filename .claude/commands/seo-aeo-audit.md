# SEO AEO audit

Audit Mercasto public visibility.

Check:
- homepage title and description
- canonical URLs
- Open Graph metadata
- Twitter card metadata
- structured data
- sitemap
- robots
- category and listing crawlability
- internal linking
- public copy clarity for Mexico marketplace intent

Output:

## Blocking issues

List issues that can hurt indexing or public visibility.

## Improvements

List safe improvements.

## Smoke

List commands or URLs to verify.

## Rollback

Explain safe rollback.

Do not promise guaranteed ranking. Focus on crawlability, metadata, content quality, trust signals, and technical correctness.
