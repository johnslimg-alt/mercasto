# Mercasto - Mexico's Local Classifieds

Mercasto — это современная и быстрая платформа объявлений для Мексики. Проект объединяет покупателей и продавцов в таких категориях как недвижимость, автомобили, электроника, работа и услуги.

Платформа включает в себя продвинутые инструменты для PRO-продавцов (дашборд, аналитика просмотров и кликов, массовая загрузка), публичную витрину компаний (Storefront) и интерактивные карты.

## 🚀 Основной функционал

- **Продвинутый поиск и фильтрация:** Поиск по ключевым словам, категориям и штатам/городам.
- **Кабинет пользователя (Dashboard):** Управление объявлениями (активация, пауза, удаление, редактирование).
- **PRO инструменты:** Аналитика просмотров и контактов (QR) с красивыми графиками на базе Recharts, массовая загрузка объявлений (CSV/XML).
- **Система рейтингов и отзывов:** Возможность оценивать продавцов от 1 до 5 звезд.
- **Умные контакты:** Генерация QR-кода для связи с продавцом без перехода по сторонним ссылкам.
- **Интерактивная карта:** Предпросмотр местоположения товаров или объектов недвижимости через Google Maps API.
- **Темная тема и i18n:** Поддержка светлого/темного режимов (Dark Mode) и мультиязычности (Испанский, Английский, Португальский).
- **OAuth Авторизация:** Поддержка входа через Google, Apple и Telegram.

## 🛠 Технологический стек

- **Frontend:** React 18, Vite, Tailwind CSS, Lucide React (иконки), Recharts (графики).
- **Backend:** Laravel 11, PHP 8.2+, MySQL.
- **Аутентификация:** Laravel Sanctum (Token-based API).

## 📦 Установка и запуск (Локально)

### Требования
- PHP >= 8.2
- Composer
- Node.js >= 18
- MySQL или другая совместимая БД

### 1. Установка Backend (Laravel)

```bash
cd backend
composer install

# Скопируйте конфигурацию окружения
cp .env.example .env

# Сгенерируйте ключ приложения
php artisan key:generate

# Настройте подключение к БД в файле .env, затем выполните миграции:
php artisan migrate

# Создайте симлинк для публичного хранилища (чтобы загружались картинки)
php artisan storage:link

# Запустите локальный сервер разработки
php artisan serve
```

### 2. Установка Frontend (React)

```bash
# В корневой директории фронтенда (где находится package.json)
npm install

# Запуск сервера разработки Vite
npm run dev
```
*Фронтенд по умолчанию будет доступен по адресу `http://localhost:5173`.*
